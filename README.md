# 基于ssm的选课管理系统3

#### 介绍
基于ssm的选课管理系统3拥有两种角色分别为教师和学生。

教师：登录、修改密码、查询个人信息、查询个人开课情况、添加、修改、删除课程、查看课程选课情况、删除学生、评分

学生：登录、修改密码、查询个人信息、查询排课情况、根据老师姓名/学院筛选、进行选课、查询个人选课情况、查询成绩
